slides.johnsonlee.io

A lightweight, CDN-hosted slide framework built on Reveal.js.

Quick Start

<head>
  <link rel="stylesheet" href="https://slides.johnsonlee.io/slides.css">
</head>
<body>
  <script src="https://slides.johnsonlee.io/slides.js"></script>
  <script>
    Slides.init({
      title: 'My Presentation',
      chapters: ['cover', 'chapter-1', 'chapter-2'],
      langs: { en: 'EN', zh: '中文' }
    });
  </script>
</body>

Features

  • Automatic Reveal.js setup (CSS + JS + plugins)
  • Multi-language support with toggle button
  • Auto-layout: centering, flex stretching, spacing
  • Dynamic page titles
  • Customizable accent color via CSS variables

CSS Variables

:root {
  --slides-accent: #e7ad52;
  --slides-accent-rgb: 231, 173, 82;
}
